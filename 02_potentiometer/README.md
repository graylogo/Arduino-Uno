# 02 - 电位器阻值读取

## 实验目的
学习 Arduino 模拟输入（ADC）读取，并理解电压分压原理

## 实验材料
- Arduino Uno 开发板
- 面包板
- 10KΩ 电位器（可调电阻）1个
- 杜邦线若干

## 硬件连接

```
Arduino 5V  ──────── 电位器 引脚1 (VCC)
Arduino A0  ──────── 电位器 引脚2 (中间抽头)
Arduino GND ──────── 电位器 引脚3 (GND)
```

电位器外观参考：
```
    ┌─────┐
  ──│     │─── 5V
    │     │
    └───│───┼─── A0
        │   │
       GND  可调中间抽头
```

## 代码说明
- `analogRead(POT_PIN)` - 读取模拟引脚的 ADC 值（0-1023）
- ADC 分辨率：10 位，即 2^10 = 1024 个等级
- 电压计算：`voltage = (adcValue / 1023.0) * 5.0`
- 阻值计算：通过电压分压公式反推 `R2 = Vout / VCC * R_total`

## 串口监视器
1. 上传代码后，打开 Arduino IDE 的"工具 > 串口监视器"
2. 设置波特率为 9600
3. 调节电位器旋钮，观察数值实时变化

## 输出格式示例
```
ADC: 512 | Voltage: 2.50V | Resistance: 5000 Ohm
ADC: 256 | Voltage: 1.25V | Resistance: 2500 Ohm
```

## 预期效果
旋转电位器旋钮，串口监视器实时显示：
- ADC 值（0-1023）
- 对应电压值（0-5V）
- 计算得到的阻值（0-10KΩ）

## 实验结果记录
**现象：** 旋转电位器旋钮时，串口显示的阻值在 0～10000 Ohm 之间连续变化。

**解释：**
1. 这表明电位器是 10KΩ 规格（B10K）
2. 硬件连接正确，ADC 读取正常工作
3. 电位器旋转时，中间抽头在两端之间滑动，改变分压比
4. 当抽头在最靠近 GND 端时，阻值接近 0 Ohm
5. 当抽头在最靠近 5V 端时，阻值接近 10000 Ohm
